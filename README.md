## Español

Pequeño gestor de tareas hecho en Python que funciona desde la terminal.

### Funcionalidades

- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
- Ver todas las tareas
- Filtrar tareas (hechas / pendientes)

Los datos se guardan en un archivo JSON.

## ENGLISH
Simple task manager written in Python that runs in the terminal.

## Features

- Create tasks
- Complete tasks
- Delete tasks
- Show tasks
- Filter tasks (done / pending)

Data is stored in a JSON file.

### Ejecutar el programa/Run
python main.py
